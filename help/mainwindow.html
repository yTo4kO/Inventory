Элементы главного окна

Главное окно программы состоит из нескольких основных элементов. Они показаны на рисунке 1.
Рисунок 1 - основные элементы главного окна
    Элементы:
  1. Главное меню
  2. Панель инструментов
  3. Выпадающий список "Группы ПК"
  4. Список компьютеров в выбранной группе
  5. Главная информационная часть - здесь отображается вся информацию о ПК
  6. Древовидное меню для выбора просматриваемого параметра
  7. Статус бар - здесь отображается информация по установленным на ПК программам и справка по некоторым элементам

Меню 'Файл'

    Меню 'Файл' содержит следующие пункты:
  1. Выход - закрытие окна и выход из программы

Меню 'Вид'

    Меню 'Вид' содержит следующие пункты:
  1. Цветовая схема - включает\отключает цветовую схему главной таблицы (пример на рис. 2)
  2. Показать скрытые - включает\отключает отображение "скрытых" программ

Рисунок 2 - сверху цветовая схема отключена, сверху включена

Меню 'Правка'

    Меню 'Правка' содержит следующие пункты:
  1. Выход - закрытие окна и выход из программы

Меню 'Отчёты'

    Меню 'Отчёты' содержит следующие пункты:
  1. Быстрый отчёт - позволяет сгенерировать отчёт по выбранному компьютеру
  2. Просмотрщик отчётов - открывает окно просмотрщика отчётов, предназначенного для просмотра сохранённых на компьютере отчётов

Меню 'Настройки'

    Меню 'Настройки' содержит следующие пункты:
  1. Пользователи - открывает окно для смены пароля пользователям

Меню 'Справка'

    Меню 'Справка' содержит следующие пункты:
  1. Содержание - открывает данную справку
  2. О программа - открывает окно 'О программе'
